تطبيق تفاعلي لتصور الشبكات والاتصالات بين نقاط الاتصال (Hotspots) مع واجهة بصرية جميلة وسهلة الاستخدام.

## ✨ الميزات الرئيسية

- 🔗 **تصور شبكات ديناميكي** برسوم توضيحية جميلة
- 📊 **تحليل الاتصالات** وعرض المعلومات الإحصائية
- 🎨 **واجهة مستخدم حديثة** وسريعة الاستجابة
- 📱 **تصميم متجاوب** يعمل على جميع الأجهزة
- ⚡ **أداء محسّن** وتحميل سريع

## 🚀 عرض مباشر

[🔗 اضغط هنا لعرض المشروع](https://network-mu-three.vercel.app)

## 💻 البدء السريع

### المتطلبات
- متصفح حديث (Chrome, Firefox, Safari, Edge)

### التثبيت المحلي

```bash
# استنساخ المستودع
git clone https://github.com/helx0/hotspot-page-.git
cd hotspot-page-

# فتح الملف في المتصفح
open deepseekNetwork.html
# أو في Windows
start deepseekNetwork.html
```

## 📖 الاستخدام

1. افتح الملف `deepseekNetwork.html` في متصفحك
2. شاهد الشبكة تتحرك وتتفاعل
3. استخدم الفأرة للتفاعل مع نقاط الاتصال

## 🎯 الميزات التقنية

- استخدام Canvas API لرسم الشبكات
- محاكاة فيزيائية واقعية للاتصالات
- تحديث فوري للبيانات
- معالجة أحداث الفأرة للتفاعل المباشر
